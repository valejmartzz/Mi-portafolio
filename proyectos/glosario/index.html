Glosario TI

A - M

A B C D E F G H I J K L M

A
Algoritmo: Conjunto de reglas y operaciones lógicas para resolver un problema.

B
Backup: Copia de seguridad de archivos o datos para evitar la pérdida de información.

C
Cloud Computing: Uso de recursos informáticos a través de internet sin depender de hardware local.

D
Debugging: Proceso de identificación y corrección de errores en un software.

E
Ethernet: Tecnología de redes de área local (LAN) que permite la comunicación entre dispositivos.

F
Firewall: Sistema de seguridad que protege una red o dispositivo de accesos no autorizados

G
GPU (Graphics Processing Unit): Unidad de procesamiento gráfico utilizada en videojuegos y diseño.

H
Hardware: Componentes físicos de un sistema informático, como el procesador y la memoria.

I
IP (Internet Protocol): Dirección única asignada a cada dispositivo conectado a internet.

J
Java: Lenguaje de programación ampliamente utilizado para desarrollo de aplicaciones.

K
Kernel: Núcleo del sistema operativo que gestiona los recursos del hardware y software.

L
Latencia: Tiempo de respuesta de un sistema o red al enviar y recibir datos.

M
Machine Learning: Rama de la inteligencia artificial que permite a las máquinas aprender de datos sin programación explícita.
