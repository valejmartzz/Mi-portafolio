Glosario TI

N - Z

N O P Q R S T U V W X Y Z

N
Network: Conjunto de dispositivos interconectados que comparten recursos y datos.

O
Open Source: Software cuyo código fuente es accesible y modificable por cualquier persona.

P
Protocol: Conjunto de reglas que definen la comunicación entre dispositivos en una red.

Q
QoS (Quality of Service): Mecanismo que prioriza el tráfico de red para garantizar un rendimiento óptimo.

R
Router: Dispositivo que dirige el tráfico de datos entre diferentes redes.

S
Server: Computadora que proporciona servicios y recursos a otros dispositivos en una red.

T
TCP/IP: Protocolo de comunicación utilizado en internet para la transmisión de datos.

U
URL (Uniform Resource Locator): Dirección única que identifica un recurso en internet.

V
Virtualization: Creación de versiones virtuales de recursos físicos, como servidores o redes.

W
Wi-Fi: Tecnología de red inalámbrica que permite la conexión a internet sin cables.

X
XML (eXtensible Markup Language): Lenguaje de marcado utilizado para estructurar datos.

Y
Yottabyte: Unidad de medida de almacenamiento equivalente a 1,024 zettabytes.

Z
Zero-Day: Vulnerabilidad de seguridad en software que es desconocida para el proveedor y no tiene solución.
